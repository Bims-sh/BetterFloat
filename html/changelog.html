1.1.0

What's new?

  • FEATURE: Sticker Prices → A new tag above the applied stickers now displays a calculated sticker price percentage. This SP% is depicted by the ratio of the item price difference to the total buff price (ask) of all stickers. This feature can be disabled in the popup (default: on).
  • POPUP: Completely reworked the logic behind the popup. It now also features a 'changelog' tab displaying the latest changes.
  • BACKGROUND: The extension is now able to detect and make use of the CSFloat API requests already sent by the website. This will allow many new possibilities in the future.
  • TECH-STACK: Building and deploying the extension is now done using webpack. This allows for a much cleaner codebase and easier development. Please refer to the new README for more information.

What's wrong?

  • If you have built the extension manually in the past, you may have to re-install the new one manually. You may want to switch to the officially published version on the Chrome Web Store or the Firefox Add-Ons Store instead.
