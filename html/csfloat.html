PRICES

Buff Price Reference

Sets the reference price for difference calculations on item prices.
Bid refers to the highest buy order price.
Ask refers to the lowest listing price. (Default: Bid)

Show FloatAppraiser

Show the FloatAppraiser values of an item in addition to the Buff price.

Show Buff Price Difference

This settings allows to switch between displaying the difference to the Buff price in absolute units and the original difference to the Steam price in percentage.

Show Buff Price Percentage Difference

Requires "Show Buff Price Difference" to be activated. Display the ratio of an items price to the Buff price in percentage. Basis (equality) will be 100%.

LISTINGS

Show Listing Age

Show the listing age in the listing card. The two options determine the position of the text. Either it is displayed as part of the header, below the condition of the item, or next to the "watchers" of the item, on top of the image of the item.

AUTO-REFRESH

Default Interval

Sets the default interval for the auto-refresh feature. Can still be adjusted on the page itself. Default value is 30s.
